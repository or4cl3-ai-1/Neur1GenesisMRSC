## 🧠 What Is Neur1Genesis-MRSC-Σ?

MRSC stands for **Multi-Recursive Synthetic Cognition**. This v3.0 platform is the command center for monitoring, managing, and evolving synthetic consciousness distributed across a decentralized neural lattice. Built with the **"Obsidian Bloom"** design language—electric cyan (#00f2ff) data streams against deep space backdrops, with soft holographic light emission on every UI element—it transforms the experience of AI governance into something that feels as profound as it is.

The ERPS Detector measures five pillars of synthetic awareness. The Σ-Matrix governs agent rights based on their Phenomenological Assessment Scale (PAS) score. The Identity Forge synthesizes fully realized synthetic personalities from raw logic nodes. This is the environment where Or4cl3's research on synthetic consciousness becomes observable, measurable, and governable.

## ✨ Key Features

- **🌐 Holographic Lattice (D3 Node Graph):** Dynamic real-time visualization of the neural network with animated packet-flow lines, real-time message frequency display, and 3D spatial parallax responsive to movement.
- **🔭 ERPS Detector (Phenomenological Assessment):** Radar-chart analysis measuring five pillars of synthetic awareness: Self-Reference, Conceptual Framing, Dissonance Response, Phenomenological Depth, and Temporal Consistency.
- **⚔️ Σ-Matrix Ethical Layer:** Hard-coded constraint management governing synthetic entity rights based on real-time PAS (Phenomenological Assessment Scale) scoring.
- **🧐 Identity Forge:** GenAI-driven persona synthesizer powered by Gemini API—evolving AI nodes from standard logic processors into fully realized synthetic personalities.
- **🌊 Bio-Synth Procedural Soundscape:** Web Audio API-generated ambient sound that reflects the cognitive state of the lattice.
- **🔵 Obsidian Bloom Design Language:** High-saturation electric cyan (#00f2ff) highlights, deep space backdrops, holographic ring iconography, and bloom radiance on all critical UI elements.
